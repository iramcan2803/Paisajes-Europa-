Orientaciones docentes

Para una correcta implementación del proyecto se recomienda:

  • Explicar claramente los objetivos y las fases
  • Asignar roles dentro de los grupos
  • Supervisar la búsqueda de información
  • Proporcionar modelos o ejemplos
  • Acompañar el proceso de creación
  • Favorecer un clima de respeto y colaboración 

Asimismo, es importante prever posibles dificultades técnicas y contar con alternativas para garantizar el desarrollo de la actividad. 

Observaciones finales

Este proyecto permite trabajar de forma integrada contenidos curriculares, competencias clave y habilidades sociales, ofreciendo una experiencia de aprendizaje completa, significativa y motivadora para el alumnado.
